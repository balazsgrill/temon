Textualmodeler

Textual modeling toolkit for Eclipse Modeling Framework

Download .zip Download .tar.gz View on GitHub

Introduction

Textualmodeler is the most simple method to create a DSL (Domain Specific Language) based on EMF (Eclipse Modeling Framework).

  • No code generation required
  • Convenient editor with syntax highlighting is available
  • More features to come..

Install

You can install Textualmodeler using the following p2 update site: http://balazsgrill.github.io/textualmodeler/site/master/
